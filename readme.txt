== Description ==

Get unlimited uptime monitoring for free! Monitor your website and get notified within minutes of any issues.

Why are we 100% free, you might ask? Read about it [here](https://tethered.app/why-free)

### Features

* Unlimited uptime checks
* Unlimited integrations
* Monitor your website
* Monitor your database or email server
* Monitor a link for a keyword or phrase

== Installation ==

= First time user =

Simply install and activate the plugin. Head over to the Tethered section and add your first monitor. It's that simple. We will let you know when your site goes down via email. If you need to be notified via webhook, Slack, Discord or any other platform, you can set that up in your account.


== Frequently Asked Questions ==

= How does this work? =

We check every "monitor" (website, database, etc) every 5 minutes and notify you when something goes wrong. It's completely free!
